# S&P 500 MCP Server Design Context

## Users
- **Primary**: Developers integrating AI tools with financial data
- **Secondary**: Financial analysts using MCP-capable AI assistants
- **Context**: Developers and analysts at desks, needing quick access to S&P 500 company data through AI assistants

## Brand Personality
- **Voice**: 高效 (Efficient)、开放 (Open)、趣味 (Fun)
- **Tone**: Clean and generous, technical but approachable
- **Emotional goal**: "This looks easy to use, I want to try it" + "Developer-friendly, easy to integrate"

## Aesthetic Direction
- **Visual tone**: Clean and generous (简洁大方), tool-documentation style like Vercel/Stripe Docs
- **Theme**: Light mode default with dark mode toggle
- **References**: Vercel, Stripe Docs, Linear — clean, functional, professional
- **SoftAurora**: Hero background, colors tuned for light theme (blue-green tones for financial trust + aurora effect)

## Design Principles
1. **Efficiency first**: Clear hierarchy, minimal cognitive load
2. **Open documentation**: Easy to scan, find what you need
3. **Playful touches**: SoftAurora adds life without being distracting
4. **Functional beauty**: No decoration for decoration's sake
5. **Dual audience**: Works for both developers (code-focused) and analysts (data-focused)
